Step 3: Combine the Functions

These two functions create the logical base for the Y Combinator. Let's combine these two functions and tentatively call it 'ycomb'. 

function helper (recurser, x) {
   return recurser(recurser, x);
}
function factorial(f, x) {
   return x < 1 ? 1 : x * f(f, x - 1);
}
helper(factorial, 5)

Turns into:

function ycomb (recurser, x) {
   return recurser(recurser, x);
}(function factorial(f, x) {
   return x < 1 ? 1 : x * f(f, x - 1);
}, 5);
