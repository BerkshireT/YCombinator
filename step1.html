Step 1: Defining the Y Combinator

The Y Combinator is a complex yet ingenious feature of functional programming languages which was developed by Haskell Curry to prove lamdba calculus is Turing complete. It is a fixed point higher-order function which aims to implement recursion in programming languages that otherwise wouldn't have it.

Here's the Y Combinator method, written in JavaScript, used on the main page to calculate each expression:

function ycomb(f) {
   return (function(g) {
      return g(g);
   })(function(g) {
      return f(function() {
            return g(g).apply(this, arguments);
      });
   });
}

This function is a lot to take in at first, so let's start with a familiar version of the recursive factorial function:

function factorial(x) {
   return x < 1 ? 1 : x * factorial(x - 1);
}

This is the basic factorial algorithm: If the number is 0, then the factorial is equal to 1; Otherwise, multiply the number by the factorial of one less than the number.
